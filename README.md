# Projeto-Ciência-de-Dados
Este projeto tem como objetivo investigar e estudar as causas da desigualdade educacional no Brasil. Serão analisados dados públicos do INEP (Instituto Nacional de Estudos e Pesquisas Educacionais), que serão  filtrados para melhor manipulação e usados para a tomada de decisão. Gestores públicos, como Secretarias de Educação, são alvos do projeto, por terem o poder de estudar os dados que nosso trabalho irá gerar e, com isso, direcionar investimentos e criar políticas públicas mais eficazes.  
A qualidade da infraestrutura escolar é um fator frequentemente citado como influente no aprendizado, entretanto ainda há a necessidade de estudar dados concretos que comprovem essa problemática. O ENEM, como avaliação nacional de grande abrangência, oferece um panorama confiável para medir esse efeito. Em suma, compreender essa relação pode auxiliar no direcionamento de investimentos e, dessa maneira, potencializar a aprendizagem dos estudantes brasileiros, principalmente em áreas que possuem um menor desempenho nesse exame.
